# NOT NULL Constraint

This example demonstrates how to use NOT NULL constraints for required fields.

## Usage

```bash
python example.py
```

## Explanation

Use `Field(..., description="not null")` to define a required field.

## Expected Output

```
Error: Name cannot be empty: ...
```
