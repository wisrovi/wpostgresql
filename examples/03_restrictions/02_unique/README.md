# UNIQUE Constraint

This example demonstrates how to use UNIQUE constraints for fields that cannot have duplicate values.

## Usage

```bash
python example.py
```

## Explanation

Use `Field(None, description="unique")` to define a field with unique values.

## Expected Output

```
Error: Duplicate email: ...
```
