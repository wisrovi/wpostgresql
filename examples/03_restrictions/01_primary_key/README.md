# Primary Key

This example demonstrates how to use Primary Key to prevent duplicate IDs.

## Usage

```bash
python example.py
```

## Explanation

Use `Field(..., description="primary")` to define a field as primary key.

## Expected Output

```
Error: Cannot insert duplicate ID: ...
```
